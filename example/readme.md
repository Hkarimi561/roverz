## Prerequisites
Install latest Node and React Native. 

## Installation 

### Step 0
```
git clone https://github.com/mongrov/roverz.git
cd mongrov/example/
```

### Step 1

```
npm install
```

### Step 2 
Start server using

```
npm start -- --reset-cache
```

### Step 3

```
react-native run-ios
```
